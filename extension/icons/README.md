# Extension Icons

This directory contains placeholder icons for the Digital Footprint Analyzer Chrome extension.

## Required Icons

The following icon sizes are required for the extension:

- `icon16.png` - 16x16 pixels (toolbar icon, small)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store, installation prompt)

## Current Status

The current icons are placeholders. For production use, replace them with properly designed icons that match the Digital Footprint Analyzer branding.

## Design Guidelines

- Use the primary blue color: #1e40af
- Include a magnifying glass or footprint symbol to represent analysis
- Ensure icons are clear and recognizable at small sizes
- Follow Chrome extension icon guidelines: https://developer.chrome.com/docs/extensions/mv3/manifest/icons/
